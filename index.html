UX DESIGN

I’ve been learning about User Experience since I was in third year of college, when I first learned about Human-Computer Interaction. I found out that UX is the core of each technology, it is bigger than just making user-friendly interface. My work as a UX designer started when I was working in Mirum Agency as a UX Designer Intern. I worked with professional to make engaging website and mobile apps. I learned and directly practice my UX knowledge through making prototype including user research, task analyzing, wireframing, and usability testing. Right now I’m doing a research about how to make good interaction design for deaf students to learn English as foreign language in mobile apps.

UI DESIGN

My experience in making user interface began in my second year of college. I had realized good user interface was somehow necessary and important to user. Long time before I knew about UX, I had been implementing on making what was good UI for user. My work as UI designer is about making mock-ups for website and mobile apps. While making mock-ups, I always think about its usability, not just using design aspects how to make them beautiful.

DEVELOPMENT

As an engineer, I do not just analyze about user. My knowledge from Informatics Engineering in Bandung Institute of Technology brings me more concept about coding. I mostly work as a front-end developer, especially in web development. I’m proficient in using HTML5 and CSS3. Right now I’m working as a Front-end Developer Intern at PT Bukalapak.com, one of the biggest online marketplace in Indonesia.

Money Snap

Money Snap is a mobile application that manage user’s daily spending. I worked with my two other friends, and I was responsible for the UX design. We decided to develop this prototype because we realized that people are lazy to manage their spending, especially teenagers. I utilized my basic knowledge about UX. I was very careful in designing this project. I wanted to create pleasurable, attractive, and wonderful apps yet was very usable and could change user’s behavior. By Money Snap, we then won 3rd Winner on UX Design Competition at Gemastik 8, National Level, in Indonesia.

Lifebuoy Apps

This is my first project as UX designer. I implemented user-centered design process to make enchanted mobile apps. Lifebuoy app is a sick-journal app that is used by moms to monitor their sick children. With this app, moms can discover why their children are infected and know what is the actual disease. This app can also monitor children’s daily life while they are sick and detect other disease around the area.

My work was to create personas based on user research, analyze the feature based on persona and feature mapping, and make wireframe through user flow, information architecture, and navigation design.

Old-books

Old-books is my first project that I utilized with my basic knowledge of Human-Computer Interaction. Old-books is a prototype of an online local bookstore that sells old college books and can only be used in specific university. The sellers and buyers are both from the same university. Other than selling feature, there are also calendar feature and chat feature.

I developed this prototype in a team consisting of 5 students. My responsibility was to design all the interface based on user requirements. I gathered the data through interview, questionnaire, and observation. Before I implemented it, I made the mockups using Adobe Photoshop.

Sharity

Sharity is a project that can make user donate to charity with a little amount of money. The idea is user can donate charity with their mobile account payment, so there’s no minimum payment for donation. This is a solution for people who are willing to donate but they don’t have much money. Albeit this is only a mobile apps prototype, this project won best 100 apps at BestApp.ID Hackathon Competition.

Lapor Taman

Lapor Taman is one of my course assignment which could help the government to control parks in my city, Bandung. This website is made to help people complaining about parks’ condition in Bandung. People can send their complains so the government will recognize the mess. Besides, this website also gives some information about how things are going in the parks.

What did I get?
In this project, my responsibilities were making user friendly interface. I worked as a front-end programmer. By finishing this project I improved my front-end skill in using HTML, CSS, and Javascript. In this project I used Bootstrap as a CSS library. I learned about Bootstrap environment and how it worked.

Link for the code: github.com/vidianindhita/Lapor-taman

Pontek's Search

This project is a search engine that can crawl several news websites. User can search some news by writing the keyword. This web crawler is a tiny version of Google. In this project I was being a front-end developer. I worked with HTML, CSS and Javascript. I didn’t use any library in this project, I developed it from scratch.

RSSep

This desktop-based application is a news reader like news feed. By using this application people can read some headline news from several news websites such as detik.com. In this project I worked as a front-end programmer. I made the interface using Java in IDE Visual Studio. I gained much knowledge in Java as developing this application.

Unilife

Unilife is a mockup of web application. This app is a social media with the concept of gamification. The idea is to encourage university student to have passion in going to class. Whenever they go to class or do the assignment they get a credit. At some point, those credits can be replaced by a badge.

Lodimana

Lodimana is an idea of mobile app that can connect your group vacation friends. I revealed this idea because sometimes we are missing our friends in a group while on group vacation. This app can detect where are all of our group members using map and location and also on the same page we can chat in a group chat. Other than those features, there are also an itinerary feature and notes feature.

MyTweetAnalytics

The Tweet Analytics is a web-based application that can search any tweet based on keyword and categories. This project use Booyer-Moore and Knuth-Morris-Prat algorithms for crawling in Java. In this project I worked in making those algorithms more effective. I also worked in making the interface to it would be user friendly.

The Simplest-blog

The Simplest Blog is a blog like Wordpress which user can login first to post their writing. In this blog user can edit and delete the post, comment on each post, and also upload photos. This blog was developed in JavaServer Page and JavaServer Faces with CSS and Javascript for the interface. In this project I focused on making the front-end from scratch.

What did I get? Since this project was my first Java web programming, I gained so much knowledge of Java web programming. I learned how Java works in developing a website rather than HTML. Besides, I created the front-end from scratch, without any templates or CSS library. I improved my skills more in front-end programming by learning CSS and Javascript.

Link for the code: github.com/vidianindhita/TheSimplestBlog
